# Neon Vengeance

A top-down action game prototype inspired by **Hotline Miami**, built with **GameMaker** using **GML (GameMaker Language)**. 

Featuring vibrant neon aesthetics, retro-synthwave animations, and fast-paced styling.

## Features

- **Interactive Main Menu**:
  - High-performance, animated selection menu with options to Start, Continue, view Options, or Quit.
  - Smooth typographic wiggling animations using custom Ease-In-Out curves.
  - Dynamic `Neon_Vengeance` logo wiggling that synchronizes with the menu text motion.
  - Full keyboard navigation (Arrow keys + Enter to select).
- **Smooth Gameplay Room**:
  - Responsive top-down WASD player movement.
  - Mouse-facing direction alignment for aiming.
  - Hit Escape (`ESC`) at any time during gameplay to return instantly to the Main Menu.
- **Aesthetic Effects**:
  - Scrolling retro palm tree background scaling effects that dynamically cycle in colors matching the synthwave hue palette.
  - Double-buffered canvas effects and customizable color gradients for that retro-arcade feel.

## Controls

| Key | Action |
| --- | --- |
| **W / A / S / D** | Player Movement |
| **Mouse cursor** | Aiming / Player Rotation |
| **Up / Down Arrows** | Main Menu Navigation |
| **Enter (Return)** | Select Menu Option |
| **Escape (ESC)** | Exit Gameplay to Main Menu |
